### Build from command line  (cd to samples/mux_mp4_remux_file)

for 32-bit build: 
	
	mkdir build32
	cd build32
	cmake -DCMAKE_BUILD_TYPE=Release32 ..
	make
	
for 64-bit build:

	mkdir build64
	cd build64
	cmake -DCMAKE_BUILD_TYPE=Release64 ..
	make

The output executable 'mux_mp4_remux_file.app' / 'mux_mp4_remux_file64.app'  goes in the SDK/lib dir.


### Build from CLion IDE

The CMake project can be opened directly in the CLion IDE.
It contains 2 targets (mux_mp4_remux_file.app and mux_mp4_remux_file64.app) with Debug and Release configurations for each target.
